# GEEK ACADEMY

Fully responsive frontend for a modern landing page for a live music/studio academy focused on converting visitors into potential students.

Tech stack:
React
HTML
CSS
JavaScript

Features:
Fully responsive
Single Page for all sections
Automic browse to sections
Functional Contact us
Video Player preview

To use, firstly pull the repository and then install the necessary node modules with 'npm install'
then enter 'npm run dev'
and the app should be running on 'localhost:5173'

enjoy.
